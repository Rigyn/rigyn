# Runtime catalog controls

Demonstrates bounded runtime discovery and explicit selection. `/example-runtime-catalog` reports active tools, available tools, registered commands, and the bounded command/prompt/skill discovery view. `/example-runtime-select` activates the first available tool, reaffirms the current model through the host, and delivers a user message as a follow-up.

These operations affect the live harness, so this package is intended for a disposable authoring workspace.

```text
rigyn install ./packages/rigyn/examples/runtime-catalog
```
